HXVM-Check host compatibility with the Virtual Machines

It can also recommend the number of VMs one can run without any problems on their host system. This program is useful for those who need to run multiple VMs for different purposes. DmonS-A framework to detect, monitor and stop broadcasted wifi deauth attack

DmonS is a framework written in python to detect the deauth packets in the network, monitor and stop the packets. This uses several python libraries such as scapy, sys, subprocess, etc. in its core. This framework works by detecting the incoming packets in the network and shows you options if it detects any deauth/disassociation packets to stop/prevent it by changing the MAC address continuously and once the packets stop coming, it reverts the MAC address to the original.
